# entertain.io

## Getting started
Clone Repo<br>
`$ git clone git@github.com:michaelzoidl/entertain.io.git`

Change Repo to **2.0**<br>
`$ git checkout 2.0`

Install Node Packages<br>
`$ npm install`

Install Bower Packages<br>
`$ bower install`

Run Gulp<br>
*Currently there's only this one task that handle the complete process.*<br>
`$ gulp build`
